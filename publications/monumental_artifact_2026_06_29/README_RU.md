публикаций и защитных границ.

Дата локального среза: 2026-06-29.

## 1. Что это

Этот узел фиксирует не новую физическую теорию и не эмпирическое
доказательство. Это навигационный паспорт монументального артефакта:
он показывает, как формальная линия TLFL, инженерная поверхность TMI-OS,
паспорта, пруф-артефакты и публикационные срезы связаны между собой.

Коротко:

```text
MonumentalArtifact :=
  TLFL formal library
  + TMI-OS / I1 engineering surface
  + guarded passports
  + publication map
  + proof-status boundary
```

Русская формула чтения:

```text
монументальный артефакт :=
  библиотека формальной проверки
  + инженерная поверхность автора
  + карта публикаций
  + граница: что доказано, что только кандидат, что только видение автора
```

## 2. Главная граница

Этот узел не повышает статус утверждений сам по себе.

```text
diagram != proof
candidate != detected particle
formal prover != empirical validation
LLM assistance != scientific authority
author visual card != public evidence
```

Формальные и инженерные артефакты могут фиксировать структуру,
проверяемость, трассу, паспорт и границы. Эмпирическое доказательство
остается отдельной внешней процедурой: данные, независимая репликация,
измерения, рецензирование и проверка внешними доменами.

## 5. Карта публикационной линии

Рабочая линия артефакта:

```text
TLFL
-> TMI-OS / I1 engineering surface
-> self-model / mathematical board passports
-> point_record / elusive graviton candidate
-> author intuition / methodology layer
-> publication and review packages
```

Назначение этой карты - удерживать связи, а не смешивать классы.
Формальная библиотека, философский трактат, визуальная доска, Zenodo-срез
и журнальная подача должны сохранять разные статусы.

## 6. Классы артефактов

| Класс | Что входит | Защитная граница |
|---|---|---|
| Formal | Lean / TLFL / OLean / TPTP / Vampire / E | Проверяет форму и статус, не заменяет эмпирику |
| Engineering | TMI-OS surfaces, boards, admin, scripts | Интерфейс и сборка, не доказательство само по себе |
| Publication | Zenodo packages, PDFs, README, metadata | Фиксация среза, не автоматический научный консенсус |

| Empirical | External datasets, replication, measurements | Требует внешних данных и независимой проверки |
